Gestión de Anuncios

Crear / Editar anuncio

Los anuncios publicados son comunicaciones oficiales. Una vez publicados, no se puede modificar el título, cuerpo ni archivo adjunto. Para corregirlos, se debe agregar una actualización o fe de erratas.

Si no se selecciona una fecha, el sistema archivará el anuncio automáticamente después de 30 días.
Los anuncios destacados no se archivarán automáticamente.

Agregar actualización / Fe de erratas

Esta actualización se agregará al final del anuncio original con fecha y hora. Los residentes serán notificados.
