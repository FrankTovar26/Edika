Configuración de Unidades

Configura la distribución de unidades del edificio. Puedes usar el modo simple para aplicar la misma cantidad en todos los pisos o el modo avanzado para personalizar cada piso y sótano.

Usa avanzado si el primer piso tiene locales o si cada sótano tiene distinta distribución.

Modo simple

Ejemplo: 4 → 101, 102, 103, 104
Solo si el edificio tiene oficinas/locales
Ejemplo: E-S1-01, E-S1-02...
Ejemplo: D-S1-01, D-S1-02...

Modo avanzado por piso y sótano

Genera una tabla editable para definir cuántos departamentos, oficinas/locales, estacionamientos y depósitos tendrá cada nivel.
